# 0002 — Curriculum arc: Projects 1→3 skill progression

- **Date:** 2026-06-23
- **Status:** active
- **Type:** curriculum design

## Context

Parent asked for two more lessons. Designed Projects 2 & 3 as a deliberate skill ladder on
top of the shared Notice→Wonder→Predict→Test→Talk loop, not as standalone activities.

## The progression (the non-obvious part)

Each project keeps the same loop but advances the *kind of prediction* Kai makes:

| # | Project | New inquiry skill | Predicts… | Difficulty |
|---|---------|-------------------|-----------|------------|
| 1 | Sink or Float? | predict→test reflex; sort; count | a **position** (up/down) | ★☆☆→★★★ |
| 2 | Will It Grow? | **fair test** (change one thing); observe **over time**; measure | a **change over days** | ★★☆ |
| 3 | Kitchen Color Lab | predict a **transformation**; multi-step; invent | what it **turns into** | ★★★→★★★★ |

## Decisions

1. **Interleaving on purpose:** Project 3 Day 3 re-uses Project 2's "fair test" (biggest-fizz
   challenge) inside a different activity. This is a retention move (interleaving > blocked
   practice), noted in-lesson.
2. **Every project still ends with a "Kai leads" day** (consistency with record 0001).
3. **Project 2 is honest about pacing:** plants don't need 30 min/day, so it's 30-min setup +
   ~10-min daily ritual + optional 30-min "while we wait" companion activities. Adjust if the
   parent wants strictly 30-min hands-on blocks.
4. **Glossary extended** (Parent Playbook) with: fair test, over time, mix, reaction, measure.
   Keep using these exact words going forward.
