🔬📖🔄 KaiSTEM

A little home learning course for a curious 3.5-year-old — and a teaching guide for the grown-up running it. Short, hands-on, joyful projects in three tracks that build one habit: wondering out loud.

Ages ~3–5 ~20–30 min/day Parent-led · screen-light

The whole idea in one line

You don't deliver facts — you narrate curiosity. Every project runs a familiar five-move loop so it feels effortless for both of you. Each track has its own loop, tuned to how that kind of thinking grows — but the spirit is identical: guess first, then find out, and joy over correctness.

Start here

New to this? Read the playbook once, then open any track below and run its first project. Print each project's journal page and keep the filled-in pages in a folder — that's the keepsake.

Built on trusted sources

The method is grounded in early-childhood research — NAEYC, HeadStart.gov, and Zero to Three across tracks; Reading Rockets for Words; and the spatial-cognition literature (Uttal 2013) for Visual. Every lesson cites its sources, and each track keeps its own RESOURCES.md. See HUB.md for how the whole space fits together.

Want to use this with your own kid? Swap in their name and interests, then run the same loops. The projects scale from ★☆☆ up — start easy and follow what lights them up.
