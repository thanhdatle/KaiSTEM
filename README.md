# 🔬 KaiSTEM

A home STEM "course" for **Kai (age 3.5)** — and a teaching guide for **you**, the parent.

> **Mission:** spark *lifelong curiosity* and *quality bonding time*.
> Joy and wonder over correctness. ([full mission](./MISSION.md))

---

## The one idea behind everything

Kai can't read these files — so **you're the learner here**. Each lesson teaches *you*
how to run a short, hands-on science project *with* Kai: what to prep, what to say, what to
look for. Kai is the scientist; you're the guide.

Every project — forever — runs the same five-move loop, so it feels familiar to Kai and
effortless for you:

> 👀 **Notice** → 💭 **Wonder** → 🤔 **Predict** → 🧪 **Test** → 💬 **Talk**

The goal is the **habit** ("guess first, then find out"), not the facts.

---

## Start here (first time)

1. **Read the [Parent Playbook](./reference/0001-parent-playbook.html)** (~3 min). The method,
   the "magic words" question bank, the glossary, and how the journal works. Print it; tape it
   inside a cupboard.
2. **Open [Project 1 — Sink or Float?](./lessons/0001-sink-or-float.html)** and gather the prep
   list the night before.
3. **Run ~30 min/day.** End each session on the **journal page** (Kai marks his guess, then what
   really happened; you scribe his exact words and date it). Keep the pages in a folder.

> Open any file in a browser: `open lessons/0001-sink-or-float.html`

## The curriculum so far

Each project keeps the same loop but advances the *kind of prediction* Kai makes:

| # | Project | New skill | Predicts… | Difficulty |
|---|---------|-----------|-----------|------------|
| 1 | [Sink or Float?](./lessons/0001-sink-or-float.html) | predict→test, sort, count | a **position** (up/down) | ★☆☆→★★★ |
| 2 | [Will It Grow?](./lessons/0002-will-it-grow.html) | **fair test**, observe over time, measure | a **change over days** | ★★☆ |
| 3 | [Kitchen Color Lab](./lessons/0003-kitchen-color-lab.html) | predict a **transformation**, invent | what it **turns into** | ★★★→★★★★ |
| 4 | [Backyard Bug Safari](./lessons/0004-bug-safari.html) | **classify** (group by attributes), close observation | how living things **group** | ★★★ |
| 5 | [Shadow Play](./lessons/0005-shadow-play.html) | **measure & compare**, spot a pattern over time | a shadow's **size/shape** & how it changes | ★★★→★★★★ |

*Coming next:* Project 6, tuned to whatever lit Kai up most. Projects 4–5 are built with
richer per-day layers (warm-up → main → extend) after we learned each day was finishing fast.

## Working with your teacher (me)

I'm here between sessions. After you run a project, **tell me how it went** — especially:

- How long did Kai actually stay engaged? (calibrates session length)
- Which part lit him up most? (steers the next theme)
- Did the "guess first" habit show up *outside* the activity?

Then I'll tune the next project to fit. You can also ask me anytime for: a re-themed project,
a harder/easier twist, different materials, or help with what to say mid-activity.
