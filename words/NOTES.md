# Notes — Words track teaching notes

Working notes for running the Words track with Kai. Written **for the parent**, about running an
activity **with Kai** (he's 3.5; he can't read these). Shared preferences and the full method live
in **[[HUB]]** (`../HUB.md`); this file keeps only what's specific to Words.

**Track accent colour:** coral (`--accent: oklch(64% 0.18 15)` / `--accent-deep: oklch(52% 0.18 16)`,
~15° hue). Science owns warm ~35°, Visual owns indigo ~295°.

## Shared conventions that apply here

- Lessons are written **for the parent**, about running an activity **with Kai**.
- Every lesson is a **multi-day arc**, each day a short (~20–30 min *ceiling*, not target) session.
- Every arc ends with a **Kai-led day** (he picks what to play with) — builds ownership.
- Build every day in **3 layers**: warm-up (~5) / main = several rounds (~15) / extend + "stretch
  it further" (~10). (Calibration from Project 1: thin days finished in ~10 min — the fix was more
  layers, not more pressure.)
- Tone: warm, concrete scripts ("say this") over theory.

## Words-specific guardrails

Distilled from [[0005-words-w2-syllables]] and hub learning record 0004 (the Words pilot):

- **Day 2 is silly listening play, not a quiz.** Never turn it into a test the child can fail.
- **Explicit permission to stop at 10–15 min.** End on a giggle, not a slog. The lesson copy says
  this out loud so the parent doesn't push for the full 30.
- **Nonsense & partial rhymes count as winning.** "Cat / zat" is a win. "Close" is a win. There is
  no wrong answer to reach for.
- **Never "no, count again."** For Word Beats, an off-by-one clap is fine — re-clap it *together*
  cheerfully rather than correcting.
- **Parent copy avoids reading-prep pressure.** No "getting ready to read" framing; the goal on the
  page is fun and connection, and the literacy payoff stays in the background.

## The Words loop tones

👂 **Listen** (tune in) · 💡 **Wonder** (out loud) · 🎭 **Play** (silly, physical) ·
✏️ **Make** (his own) · 💬 **Share** (say it, mark the journal). Movement is the engine — clap,
march, drum, jump the sounds; the body cements what the ear hears.

## Open questions to revisit

- **Did Kai rhyme / clap words out *unprompted*, away from the activity?** That transfer is the real
  signal the track is working — watch for it in daily life, not in the session.
- **What pulls next — W3 first-sounds ("ball starts with /b/", the documented next rung) or a
  storytelling turn?** Let Kai's pull decide.
